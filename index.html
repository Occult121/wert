Types of attributes
nominal attributes --- data order is not important though important is difference in data intervals
Ordinal attributes ---- important is data order though not important is difference in data intervals examples ---- socio economic status
Interval attributes ----- important is data order + difference in data intervals
Ratio attributes --- important is data order , difference in data intervals and a 0.0 where 0.0 = none or no data . examples ---- temperatures etc
